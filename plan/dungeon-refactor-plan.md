# 地牢模式重构计划

## 目标
全面重构地牢模式，实现：
1. 35-40节点随机地图生成（9场最短路径到Boss）
2. 问号迷雾机制（只能看到相邻/已走过节点）
3. 5种随机事件节点
4. 随机战斗场地（1024-2048）
5. Boss战（大块头，4096场地）
6. 奖励节点（复用雪地奖励）

## 关键设计决策

### 节点类型
- `start`: 起点
- `combat`: 战斗节点（70%）
- `event`: 随机事件（30%）
- `boss`: Boss战
- `reward`: 奖励节点（Boss后固定）
- `empty`: 已完成战斗节点（无效）

### 迷雾机制
- 未访问节点：显示"?"
- 已访问节点：显示实际类型
- 相邻节点：显示实际类型

### 随机事件类型
1. 女神像（恢复/祝福/奖励）
2. 陷阱（解除/跨越）
3. 补给堆（搜寻/探查）
4. 宝箱（金币/材料/战斗）
5. 恶魔雕像（Buff/奖励）

### 战斗场地生成
- 大小：1024-2048随机
- 玩家生成：四边中间位置随机
- 怪物生成：对边位置
- 完成后删除场地

### 属性检定公式
- 成功率 = 基础成功率 + 属性值×1%
- 上限95%，下限5%

### Buff系统
- 女神祝福：+15%物攻/魔攻，3场战斗
- 恶魔祈祷：+33%物攻/魔攻，永久（但扣血）
