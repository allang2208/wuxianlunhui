# V0.040 版本开发进度总结

- **版本号**: V0.040

## 已完成任务

### 1. 近战武器词条（V0.040-1）
- 为所有近战武器添加 `weaponTypeTag: '近战武器'`
- 武器列表：生锈的长剑(weapon1)、骑士长剑(weapon2)、符文长剑(weapon4)
- 浮窗/图鉴同步显示

### 2. 冲刺攻击技能（V0.040-2）
- **技能名称**: 冲刺攻击
- **图标**: 💨
- **等级**: 1-20级
- **触发条件**: 长按Shift冲刺≥1500ms + 左键 + 近战武器
- **动画时长**: 1200ms（三段式）
  - 0-350ms: 旋转90°（以主角为中心）
  - 350-800ms: 前伸30px + 扇形180°挥砍
  - 800-1200ms: 牵引复位
- **伤害**: 1.75×当前武器攻击力（随等级提升）
- **攻击范围**: 100px半径，180°扇形，150ms判定窗口
- **体力消耗**: 20点
- **升级方式**: 击中+1，2人以上+3，击杀+15

### 3. 技能栏扩展
- 新增冲刺攻击技能卡片
- 与剑精通并列显示
- 详情面板支持两种技能格式切换
- 经验系统：`addDashExp()` 方法

### 4. 攻击范围显示
- 冲刺攻击支持左下角"显示范围"按钮
- 白色扇形范围指示（100px，180°）
- 600ms持续时间

### 5. weapon4 粒子特效（V0.040-3）
- 粒子大小: 0.3-0.5px
- 生成位置: 剑柄（±4px）
- 待机状态: 屏幕正上方上浮（不随主角朝向改变）
- 移动状态: 移动方向反方向上浮
- 生命周期: 800-1200ms
- 透明度: 最大50%
- 三层绘制: 主粒子 + 火焰光晕 + 核心亮点

### 6. 浮窗边界检测优化
- 动态检测浮窗实际高度
- 自动调整水平和垂直位置
- 超出屏幕时自动上移/左移
