# Phaser 完整迁移计划（v0.193 → v0.200）

## 当前状态（v0.193）

| 模块 | 渲染方式 | 状态 |
|------|----------|------|
| 角色（player） | Phaser Sprite + Canvas 武器/特效 | 角色已迁移，武器在 Canvas |
| 相机系统 | Phaser Camera | ✅ 完成 |
| 墙壁碰撞 | Phaser ArcadePhysics + WallSystem | ⚠️ 部分完成 |
| 敌人（14种） | 13种 Phaser，1种 Canvas | 接近完成 |
| 树木/环境 | Canvas | ❌ 未迁移 |
| 粒子特效 | Canvas | ❌ 未迁移 |
| 开发工具 | Canvas 预览 + 混合配置 | ❌ 需重写 |

**核心问题**：武器渲染在 Canvas 和 Phaser 之间来回切换，坐标变换链不一致导致位置错位。解决方案：一次性完整迁移武器到 Phaser，删除所有 Canvas 武器渲染代码。
